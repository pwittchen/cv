# resume
my resume (cv)

## installing required software
- ubuntu: `sudo apt-get install texlive`
- arch: `sudo pacman -S texlive-most`

## usage
- to create `*.pdf` document, type: `make pdf`
- to clean generated documents type: `make clean`
